# 🚩 CTF Workspace: The Raw Logs

> "If I didn't write it down, did the flag even exist?"

Welcome to my central repository for all things **Capture The Flag**. This is my digital scratchpad—the raw, unedited, and sometimes caffeinated notes I take while hacking through machines and challenges.

---

## 🧠 Purpose & Intent
The goal of this workspace is **persistence over perfection**. I use this space to:

* **Document the "Aha!" Moments:** Capturing the exact command that finally worked.
* **Build a Knowledge Base:** A searchable archive of exploits, payloads, and bypasses.
* **Track Progress:** A transparent look at my journey through the world of cybersecurity.

---

## 📂 Repository Structure
I categorize my findings by discipline. Inside each folder, you'll find `.md` files or raw text dumps:

| Category | Focus |
| :--- | :--- |
| **🌐 Web** | SQLi, XSS, SSRF, and breaking logic. |
| **🔓 Pwn** | Buffer overflows, heap exploitation, and shellcoding. |
| **🔍 Rev** | Ghidra sessions, assembly breakdowns, and logic reversing. |
| **🔐 Crypto** | RSA mishaps, XOR logic, and math-heavy challenges. |
| **🕵️ Forensics** | Memory dumps, PCAP analysis, and steganography. |
| **📝 Writeups** | Full walkthroughs for specific machines (HTB, THM, etc.). |

---

## ⚠️ The "Unedited" Disclaimer
**Read at your own risk.** These notes are uploaded "as-is." This means:
1.  **Typos abound:** Most of this was typed while a timer was ticking.
2.  **Chaotic logic:** Some notes might end abruptly because I found the flag and moved on.
3.  **No Hand-holding:** These are my internal logs, not necessarily step-by-step tutorials.
